

# 將Hour of Code介紹給你的學校

## 讓每個學校參予，每個學生學習！

  1. **招募當地的學校。** 分享 [這封電子郵件](<%= hoc_uri('/resources#sample-emails') %>)和[教師的指引手冊](<%= hoc_uri('/resources/how-to') %>), 或是將 [簡介](<%= hoc_uri('/resources/stats') %>)納入新聞或是當地的通訊刊物.

  2. **為學校提供一個物流計畫樣本。**分享這個物流計畫樣本給學校，讓他們從當中獲得如何組織全校共同參與的想法。 在每一堂數學課，固定教室授課期間，亦或是跑堂課程，這個計畫都可以很容易得透過電腦教室完成。 他們可以上傳他們自己的計畫，並有機會為自己贏得10,000 美元的硬體設備 [ http://hourofcode.com/prizes](<%= hoc_uri('/prizes') %>).

  3. **分享在社群媒體** 發文在[臉書](https://www.facebook.com/sharer/sharer.php?u=http%3A%2F%2Fhourofcode.com%2Fus)或[推特](https://twitter.com/intent/tweet?url=http%3A%2F%2Fhourofcode.com&text=I%27m%20participating%20in%20this%20year%27s%20%23HourOfCode%2C%20are%20you%3F%20%40codeorg&original_referer=https%3A%2F%2Fwww.google.com%2Furl%3Fq%3Dhttps%253A%252F%252Ftwitter.com%252Fshare%253Fhashtags%253D%2526amp%253Brelated%253Dcodeorg%2526amp%253Btext%253DI%252527m%252Bparticipating%252Bin%252Bthis%252Byear%252527s%252B%252523HourOfCode%25252C%252Bare%252Byou%25253F%252B%252540codeorg%2526amp%253Burl%253Dhttp%25253A%25252F%25252Fhourofcode.com%26sa%3DD%26sntz%3D1%26usg%3DAFQjCNE1GLTUbKZfMlEh9Aj5w0iswz6PYQ&related=codeorg&hashtags=). 或分享一則發人深省的 [ 圖片和世界各國領導人名言](<%= hoc_uri('/resources#social') %>)，或他們的[ 狀態](<%= hoc_uri('/resources/stats') %>)).

  4. **在推廣你分區的主頁。**加入一些[HTTP://hourofcode.com](<%= hoc_uri('/resources#banners') %>橫幅</a>在你的分區網頁來讓參觀者瞭解你的參與。 將其連結到 <a href=).

  5. **主辦小時的代碼分區事件。**看到我們 [ 事件如何引導](<%= hoc_uri('/resources/how-to-events') %>) 作為示例而顯示運行，媒體會聯絡工具組和其他支援。

  6. **更多 [ 獎品](<%= hoc_uri('/prizes') %>) 為每個召集人.**每個參與的教育家會收到謝謝你的禮物。 同時，美國的每州，所有參與的學校將會為他們的學校贏得 $10,000 美元的科技產品！
