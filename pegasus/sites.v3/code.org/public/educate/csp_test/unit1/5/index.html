Lesson 5: Binary Numbers

Overview

In this lesson, students will gain more familiarity with binary numbers. The lesson will transition away from the number systems that students created in the the circle-triangle-square activity, and begin to focus on representing numeric values using the binary number system. Though students have communicated with Binary before, developing a number system is a little different. Previously, students mapped patters of binary values to a small, fixed set of messages. A number system is infinite, and also has rules for counting - or how to get from one value to the next.

Purpose

Number systems help us express and reason about quantities. Early number systems were merely a system of tallies that allowed humans to record and perform simple arithmetic with values. The number system we use today uses the concept of place value to allow us to express any value we wish by combining only 10 symbols (0, 1, 2 …). We therefore call it a “base 10” number system. When developing a number system for a computer, we only have two symbols available to us, corresponding with the two states of a single bit. However, the power of place value allows our binary or “base 2” number system to express any value we wish.

When using this binary representation of numbers, certain values (1, 2, 4, 8, 16, etc.) are seen repeatedly. When written in binary, these values are 1, 10, 100, 1000, 10000, and so on, and so are the incremental place values in this binary number system.

Objectives

Students will be able to:

Teaching Guide

Getting Started

Recap: Number Systems: Circle-Triangle-Square activity and properties of number systems

Discussion Goal

In the previous lesson students designed number systems by making permutations of colored shapes and defining rules about place value. This warm-up presents students with only two symbols they need to use in building their permutations, as well as a set of rules for ordering them. They will become quite familiar with this particular set of permutations and rules as they define the binary number system, the tools computer scientists use to represent numbers in bits.

Exploration: Distribute the day’s Activity Guide - Binary Practice to students. Ask them to read the directions or do so as a class before giving them time to develop the binary representations of the values 0-7. It may be necessary to demonstrate the first couple of values.

Discuss: Ask students to compare their results with their classmates’ and clear up any discrepancies together or with the help of the class. Then lead a discussion around the following prompts:

  • Why might we want to create a number system that includes only two symbols?
    • Then we can use bits as the building blocks of our number system.
  • How large of numbers do you think this system can represent? How could we go higher?
    • Adding more bits (increasing the size of our permutations) allows us to count as high as we wish.

Transition: The number system we have just been introduced to is called the binary number system and can be constructed entirely from bits. This is the number system implemented in almost every computer. While it may look different from our familiar number system, as we’ll see in today’s lesson, it can be used in the same way and shares many properties.

Activity

Construct a Flippy Do

Say: For today’s activity you will be creating your own “Flippy Do.” This is a tool that will allow you to quickly and easily translate between decimal numbers and binary numbers, or in other words, between the numbers we use every day and bits.

Instructions

  • To get started, cut along the dashed lines to create individual tabs on the bottom of your Flippy Do.
  • In the third row (right above the tabs) add a 0 to each square.
  • Flip up your tabs and write a 1 on the back of each of them.

Use the Binary Odometer Widget to Check Work

Say: We’ll keep filling in our Flippy Do’s but we need some information first. Go to the Binary Odometer. This tool allows you to watch how counting in binary numbers is related to counting in decimal numbers. Using the binary odometer, list all 8-bit binary numbers that have exactly one “1” in them, along with their decimal equivalents. There’s space provided on your worksheets.

  • Students should work in pairs or groups of 3. Give students a few minutes to get acquainted with the Binary Odometer and fill in this information on their worksheets.

Say: What patterns do you observe in these values? Why might they exist?

  • The numbers that have exactly one “1” are all powers of 2 (1, 2, 4, 8, 16, 32, etc.) Compare to base 10 (1, 10, 100, 1000) where every number with one “1” in it is a power of 10.

Complete Flippy Do: Lead students through filling in this information as shown below, completing their Flippy Do’s.

Teaching Tips

It may be necessary to demonstrate how values can be calculated by flipping up a “1” for each value required to arrive at the sum of values equal to the decimal number.

If students are having a difficult time understanding the rules of the system, remind them of the concept of place value and relate to base 10.

Complete the Binary Practice Activity Guide

Direct students to complete the activity guide individually using the Flippy Do and Odometer as resources. Students are encouraged to work with others, but should each complete the worksheet. If students finish early, direct them to the Cisco Binary Game, an applet that requires them to translate between base 10 and binary numbers. Students can find a link to the applet in Code Studio.

Wrap-up

Discuss Students’ Completed Activity Guides

Open up the floor to respond to any lingering questions that may have developed over the course of today’s activity.

Alternative: students may spend this time completing the two reflection questions on Code Studio.

Extended Learning

  • How much memory does your music player, phone, or computer have? Do you notice anything related to binary?
  • Practice translating between bases using the Cisco Binary Game.

Assessment

Suggestions

  • The Activity Guide can be assessed, or generate similar questions asking students to translate back and forth between decimal and binary. Encourage students to try to do so without the assistance of their Flippy Do once they become more comfortable.
  • Prompt (written): Explain why this joke is funny: “There are 10 kinds of people in the world, those who understand binary and those who don’t.”
