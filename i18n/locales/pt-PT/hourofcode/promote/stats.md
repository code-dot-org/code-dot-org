### Bring computer science to your school. Start with an Hour of Code!

Com a tecnologia a mudar todas as indústrias no planeta, o conhecimento computacional tornou-se parte de um conjunto de capacidades globais. But fewer than half of all schools teach computer science! Good news is, we’re on our way to change this. If you heard about the Hour of Code last year, you might know it made history. Mais de 100 milhões de estudantes já descobriram o quão acessível a ciência computacional pode ser, através de apenas uma Hora de Código.

The Hour of Code is a one-hour introduction to computer science, designed to demystify code and show that anybody can learn the basics. Learn more at <http://HourofCode.com>, try an hour yourself, or host an Hour of Code event to introduce others to the world of computing!
