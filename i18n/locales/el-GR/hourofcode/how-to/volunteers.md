# Με ποιο τρόπο να συμμετέχετε εθελοντικά σε ένα σχολείο ή μια τάξη

### Δεκάδες χιλιάδες εκπαιδευτικοί αναζητούν εθελοντές για να εμπνεύσουν τους μαθητές τους. Κάνε τη διαφορά σήμερα!

[<button>Εγγραφείτε για να είστε εθελοντές</button>]({{ urls/volunteer }})

* * *

Η υπολογιστική σκέψη συμβάλλει στην ανάπτυξη δεξιοτήτων επίλυσης προβλημάτων, λογικής, και δημιουργικότητας. Και η τεχνολογία μεταμορφώνει κάθε βιομηχανία πάνω στον πλανήτη. Οι μαθητές σήμερα θα πρέπει να μάθουν τον τρόπο δημιουργίας τεχνολογίας, όχι μόνο τη χρήση της.

Η Ώρα του Κώδικα είναι μια μονόωρη εισαγωγή στην επιστήμη της πληροφορικής σχεδιασμένη να απομυθοποιεί τον "κώδικα", να δείξει ότι οποιοσδήποτε μπορεί να μάθει τα βασικά, και να διευρύνει την συμμέτοχη στον τομέα της επιστήμης των υπολογιστών. Μπορείτε να συμβάλλετε στην ενημέρωση της κίνησης της επιστήμης των υπολογιστών και να συμμετέχετε εθελοντικά για να εμπνεύσετε περισσότερους μαθητές να δοκιμάσουν την επιστήμη της πληροφορικής, ειδικότερα νεαρές γυναίκες και μαθητές από ιστορικώς περιθωριοποιημένες φυλετικές και εθνικές ομάδες.

<a id="who-can-volunteer"></a>

## Ποιος μπορεί να γίνει εθελοντής;

Οποιοσδήποτε έχει πάθος με την εκπαίδευσή πάνω στην επιστήμη της πληροφορικής και με την αύξηση της ποικιλομορφίας στην τεχνολογία μπορεί να είναι εθελοντής! Θα θέλαμε να δούμε εθελοντές να συμμετέχουν από όλα τα υπόβαθρα. Η Ώρα του Κώδικα χαρακτηρίζεται από μια μεγάλη ποικιλία δραστηριοτήτων για όλες τις ηλικίες και τα επίπεδα δεξιοτήτων, συνεπώς δε χρειάζεται να είστε ένα ειδήμονας στον προγραμματισμό για να είστε εθελοντής!

Μπορείτε να είστε ένας προσκεκλημένος ομιλητής σε μια τάξη ή να βοηθάτε μια δασκάλα να πραγματοποιήσει μια δραστηριότητα. [Εγγραφείτε σήμερα](https://code.org/volunteer) για να εμπνεύσετε τους μαθητές να συνεχίζουν να μαθαίνουν την επιστήμη της πληροφορικής.

<a id="why-volunteer"></a>

## Γιατί να γίνω εθελοντής;

Η επιστήμη της πληροφορικής είναι το πεδίο που ορίζει τον 21ο αιώνα, μολαταύτα πολλά σχολεία ακόμη δεν τη διδάσκουν. Μια Ώρα του Κώδικα ενδέχεται να είναι η μέγιστη πρόσβαση στην επιστήμη των υπολογιστών που έχουν αυτοί οι μαθητές σε ένα συγκεκριμένο έτος. Όντας εθελοντής, μπορείτε να διευκολύνετε ακόμη περισσότερο τους εκπαιδευτικούς να φέρουν την Επιστήμη της Πληροφορικής μέσα στην τάξη τους.

[![](/images/fit-600/Marketing/HoC-2015-ACE-HS--42.jpg)]({{ urls/learn }})

Διαβάστε μερικές εμπειρίες προηγούμενων εθελοντών:

- «Το καλύτερο κομμάτι της εμπειρίας μου ήταν πως όλη η τάξη ήταν έγχρωμα κορίτσια. Είναι καλό για το μέλλον της βιομηχανίας μας να έχει την ευκαιρία να προσεγγίσει και να ενθουσιάσει αυτά τα παιδιά.»
- «Χάρηκα ιδιαίτερα που μου δόθηκε η ευκαιρία να ενθαρρύνω τα νέα κορίτσια προς την τεχνολογία. Αισθάνθηκα πως βοήθησα την νέα γενιά στην περιοχή μου.»
- "ΛΑΤΡΕΨΑ το πόσο ενθουσιασμένα ήταν τα παιδιά. «Ήταν καταπληκτικό ότι οι εκπαιδευτικοί παρατηρούσαν πως κάποιοι μαθητές, οι οποίοι δεν απέδιδαν ιδιαίτερα καλά στα σχολικά μαθήματα, αρίστευαν στον τρόπο σκέψης σαν προγραμματιστές και βοηθούσαν άλλους - δίνοντας τους ένα μέρος για να νιώθουν σιγουριά."

<a id="selection"></a>

## Με ποιο τρόπο επιλέγονται οι εθελοντές;

Οι εκπαιδευτικοί μπορούν να αναζητήσουν εθελοντές στο δικό μας [χάρτη εθελοντών](http://code.org/volunteer/local). Αν βρίσκεστε κοντά στην τάξη τους, οι εκπαιδευτικοί θα εξετάσουν το προφίλ σας στο χάρτη, για αυτό προσπαυήστε να συμπληρώσετε όσο το δυνατόν περισσότερα για να αυξήσετε την πιθανότητα μια εκπαιδευτικός να επικοινωνήσει μαζί σας.

[![](/images/fit-600/Marketing/2018_HoC-489-resized.jpg)]({{ urls/learn }})

Όταν επιλεγείτε, μια εκπαιδευτικός θα επικοινωνήσει μαζί σας μέσω της πλατφόρμας εθελοντών (δε θα κοινοποιήσουμε ποτέ τη διεύθυνση ηλεκτρονικού ταχυδρομείου σας με τον εκπαιδευτικό). Εργαστείτε μαζί τους για να καθορίσετε με ποιο τρόπο μπορείτε να συνεισφέρετε καλύτερα στην εκδήλωσή τους, και αν θα τους επισκεφθείτε αυτοπροσώπως ή θα είστε εθελοντής εικονικά.

Αν λαμβάνετε υπερβολικά πολλά αιτήματα από δασκάλους, μπορείτε πάντα να ενημερώνετε τις προτιμήσεις σας ή να απεγγραφείτε κάνοντας κλικ στον σύνδεσμο που παρέχεται στο κάτω μέρος οποιουδήποτε αιτήματος μέσω μηνύματος ηλεκτρονικού ταχυδρομείου από μια δασκάλα.

<a id="choose-a-school"></a>

## Τι θα συμβεί αν θέλω να συμμετάσχω εθελοντικά σε ένα συγκεκριμένο σχολείο;

Δε χρειάζεται να χρησιμοποιήσετε το χάρτη εθελοντών μας προκειμένου να συμμετέχετε εθελοντικά! Στην πραγματικότητα, προτείνουμε να απευθυνθείτε σε δασκάλους και διευθυντές απευθείας αν υπάρχει μια τάξη στην οποία θέλετε να συμμετάσχετε εθελοντικά.

Για να ξεκινήσετε: - Εάν είστε γονέας ή κηδεμόνας, στείλτε [αυτό το email στον δάσκαλ](https://hourofcode.com/us/promote/resources#help-schools) ο του παιδιού σας προσφέροντας να το βοηθήσετε να εκτελέσει μια Ώρα Κώδικα. - Αν έχετε στο μυαλό σας ένα άλλο σχολείο, επισκεφθείτε τον ιστοτοπό τους για να βρείτε τις κατάλληλες επαφές για να απευθυνθείτε όπως διευθυντής ή υποδιευθυντής, καθηγητές τεχνολογίας ή επιστήμης της πληροφορικής, ακόμα τον Σύλλογο Γονέων και Κηδεμόνων του σχολείου. Μπορείτε να συνεργαστείτε επίσης με οργανισμούς όπως τα Boys & Girls Clubs της Αμερικής, το Σωματείο Επιχειρηματικότητας Νέων, ένα παράστημα της ΧΑΝ, και πολλά άλλα για να φιλοξενήσετε μαζί μια εκδήλωση. - Αν δεν μπορείτε να βρείτε ένα σχολείο που φιλοξενεί μια Ώρα του Κώδικα, μπορείτε να βοηθήσετε να πραγματοποιηθεί η δική σας εκδήλωση μέσω της εταιρείας σας. Ρίξτε μια ματιά στο δικό μας [Οδηγό Κατευθύνσεων για Εταιρείες](https://hourofcode.com/us/how-to/companies) για να μάθετε περισσότερα και να εμπνευστείτε.

<a id="how-to-prepare"></a>

## Πώς να προετοιμαστείτε

Είτε φροντίζετε μια εκδήλωση Ώρας του Κώδικα απομεμακρυσμένα ή αυτοπροσώπως, θα έχετε μια ομαλότερη εμπειρία αν κάνετε αυτές τις προετοιμασίες προκαταβολικά.

[![](/images/fit-600/Marketing/4Q9A5575.jpg)]({{ urls/learn }})

### Συναντηθείτε με τον Διοργανωτή

Αν συμμετέχετε εθελοντικά με μια τάξη, συζητήστε αυτούς τους συλλογισμούς με τη δασκάλα νωρίτερα:

- Το ρόλο σας στην εκδήλωση Αν εργάζεστε στον τομέα της τεχνολογίας, μπορεί να σας ζητηθεί να μοιραστείτε τις προσωπικές εμπειρίες σας ή την ιστορία σας: Τι σας οδήγησε σε μια καριέρα στην τεχνολογία; Γιατί είναι σημαντική η επιστήμη της πληροφορικής για εσάς; Ποια είναι κάποια πράγματα που κάνετε ως μέρος της εργασίας σας και πώς αυτά σχετίζονται με την τεχνολογία;
- Αν είστε εθελοντής αυτοπροσώπως, βεβαιωθείτε ότι κατανοείτε και οι δύο της τοπικές και τις σχολικές κατευθυντήριες γραμμές υγείας και ασφάλειας. Αναλόγως τις διαδικασίες τους, ενδέχεται να χρειάζεται να φτάσετε κάπως νωρίς.
- Αν θα είστε εθελοντής εικονικά ή αυτοπροσώπως καθώς αυτό μπορεί να καθορίσει τις ώρες παρουσίας σας και με ποια ιδιότητα.
- Αν θα είστε εθελοντής εικονικά, δουλέψτε μαζί με την καθηγήτρια για να καθορίσετε την καλύτερη πλατφόρμα συνδιασκέψεων βίντεο για χρήση και προσπαθήστε να διεξάγετε μαζί μια δοκιμαστική λειτουργία της εκδήλωσης. 
- Αν η δασκάλα έχει προκαθορίσει τις δραστηριότητες της Ώρας του Κώδικα για την εκδήλωση, να είστε ενήμερος έτσι ώστε να μπορείτε να [τις εξερευνήσετε](https://hourofcode.com/us/learn) νωρίτερα.

### Αναλογιστείτε τους συμμετέχοντες στην εκδήλωση σας

- Συλλογιστείτε τους μαθητές στους οποίους θα μιλήσετε και τι θα βρει ανταπόκριση σε αυτούς. Όντας εθελοντής, μπορείτε να τους βοηθήσετε να συνειδητοποιήσουν ότι η υπολογιστική σκέψη είναι παντού και μπορούν και αυτοί να γίνουν επιστήμονές υπολογιστών και πληροφορικής, επίσης. Συλλογιστείτε την ηλικία τους, το υπόβαθρό τους και τις δικές τους ταυτότητες φύλλου και ελάτε προετοιμασμένοι με θέματα που μπορεί να άπτονται των ενδιαφερόντων τους. Εδώ σας παραθέτουμε μια επιπρόσθετη καθοδήγηση σχετικά με τον τρόπο να εμπνεύσετε [νεαρές γυναίκες](http://code.org/girls) να συμμετέχουν.
- Προκαλέστε ενθουσιασμό στους μαθητές! Δουλέψτε μαζί με την δασκάλα να διαλέξετε ένα [εμπνευσμένο βίντεο](https://hourofcode.com/us/promote/resources#videos) μπορεί να συμβάλλει στη συγκέντρωση της τάξης και να ενθουσιάσει τους μαθητές να κάνουν μια Ώρα του Κώδικα.
- Αν είστε ένας εικονικός εθελοντής, σκεφτείτε να κάνετε μερικές παρουσιάσεις διαφανειών ως μέρος της παρουσίασής σας να για να κρατήσετε αμείωτο το ενδιαφέρον των μαθητών.

Χρειάζεστε περισσότερους πόρους; [Ρίξτε μια ματιά σε αυτές τις άλλες μικρές συμβουλές](https://code.org/files/CSTT_Volunteers.pdf).

## Όταν φιλοξενείτε μια εκδήλωση

Αν είστε ο οικοδεσπότης μιας εκδήλωσης που αφορά σε τάξη, χρησιμοποιήστε το παραδειγματικό πρόγραμμα να σας καθοδηγήσει. Επιβεβαιώστε το συμφωνημένο πρόγραμμα για την ημέρα και συζητήστε οποιαδήποτε τυχόν θέμα προς συζήτηση με την καθηγήτρια προκαταβολικά. Αν επισκέπτεστε σωματικά την τάξη, βεβαιωθείτε ότι καταγράφετε την άφιξή σας υπογράφοντας στα βιβλίο επισκεπτών του σχολείου και ότι καλωσορίζετε τη δασκάλα την ημέρα της εκδήλωσής σας.

Αν σκέφτεστε να φιλοξενήσετε μια Ώρα του Κώδικα στο σπίτι ή την κοινότητα, σας προτείνουμε να ρίξετε μια ματιά σε κάποιους από τους εμπεριστατωμένους και εκτενείς Οδηγούς μας Καθοδήγησης για [γονείς](https://hourofcode.com/us/how-to/parents) και [οργανισμούς ή εταιρείες](https://hourofcode.com/us/how-to/companies).

#### Άλλες ιδέες για να προσθέσετε στην εκδήλωσή σας

- Εξηγήστε τους τρόπους με τους οποίους η τεχνολογία επηρεάζει τη ζωή μας, με παραδείγματα που οι μαθητές από όλα τα υπόβαθρα θα ενδιαφέρονται - μιλήστε για την τεχνολογία που σώζει ζωές, βοηθάει ανθρώπους, συνδέει ανθρώπους.
- Αν είστε μια εταιρεία τεχνολογίας, λάτρης της επίδειξης, συζητήστε καινοτόμα προϊόντα που η εταιρεία σας εργάζεται πάνω σε αυτά. Αν δεν είστε μια εταιρεία τεχνολογίας, συζητήστε τρόπους που η εταιρεία σας χρησιμοποιεί την τεχνολογία να επιλύσει προβλήματα και να πετύχει στόχους.
- Καλέστε προγραμματιστές από την εταιρία σας να μιλήσουν σχετικά με τους λόγους που τους οδήγησαν να μελετήσουν την επιστήμη των υπολογιστών και για τα έργα στα οποία εργάζονται.
- Εάν έχετε μια ομάδα προσωπικού που θα διευκολύνει την εκδήλωσή σας ή θα κάνει εθελοντισμό με ένα τοπικό σχολείο, σκεφτείτε να παραγγείλετε [προσαρμοσμένα μπλουζάκια](https://www.amazon.com/stores/Code/page/8557B2A6-EBF2-4C9F-95C5-C3256FBA0220?ref_=ast_bln) για την ομάδα σας.

### 3. Προγραμματίστε τις τεχνολογικές σας ανάγκες

[![](/images/fit-600/Marketing/Excel-Charter-SchoolHoC-2015-stills-9.jpg)]({{ urls/learn }})

#### Συσκευές:

Για την καλύτερη εμπειρία στην Ώρα του Κώδικα, απαιτούνται υπολογιστές που συνδέονται στο διαδίκτυο. Αλλά δεν χρειάζεστε υπολογιστή για κάθε παιδί, ενώ μπορείτε να κάνετε την Ώρα του Κώδικα χωρίς καν να έχετε υπολογιστή! Για δραστηριότητες ανεμπόδιστες από την έλλειψη υπολογιστών ή συσκευών, απλώς εφαρμόστε το φίλτρο στην ενότητα Τεχνολογία Τάξης να δείχνει επιλογές για "Χωρίς υπολογιστές ή συσκευές"

- **Πραγματοποιήστε δοκιμαστική χρήση των δραστηριοτήτων** σε υπολογιστές ή τις συσκευές. Βεβαιωθείτε ότι δουλεύουν σωστά σε προγράμματα περιήγησης με ήχο και βίντεο.
- **Δεν έχετε αρκετές συσκευές;** Χρησιμοποιήστε τον προγραμματισμό ανά ζεύγη. Όταν οι μαθητές συνεργάζονται, βοηθούν ο ένας τον άλλο και να βασίζονται λιγότερο στον εκπαιδευτικό. Καταλαβαίνουν επίσης ότι η Επιστήμη των Υπολογιστών είναι κοινωνική και συνεργατική.
- **Να παρέχετε ακουστικά κεφαλής** για τους συμμετέχοντες στις εκδηλώσεις σας ή ζητήστε τους να φέρουν τα δικά τους αν θα παρακολουθούν εκπαιδευτικά βίντεο που δουλεύουν καλύτερα με ήχους.
- **Έχετε αργή σύνδεση στο δίκτυο;** Δείξτε τα βίντεο στην αρχή της εκδήλωσης, ώστε να μη χρειάζεται να κατεβάζει τα βίντεο ο κάθε συμμετέχων ξεχωριστά. Ή δοκιμάστε τις δραστηρίοτητες που δεν απαιτούν σύνδεση στο διαδίκτυο / τις εκτός σύνδεσης δραστηριότητες.

#### Εικονικές Εκδηλώσεις

Αν η εκδήλωσή σας πρόκειται να είναι εικονική, θα θέλετε να αποφασίσετε σχετικά με την πλατφόρμα συνδιάσκεψης (και να πραγματοποιήσετε δοκιμαστική χρήση) πριν από την εκδήλωσή' σας. Η πλατφόρμα συνδιασκέψεων μπορεί να επηρεάσει τον αριθμό των μαθητών με τους οποίους θα αισθάνεστε άνετα να ασχοληθείτε, άρα είναι καλύτερα να αποφασίσετε πάνω σε αυτό το ζήτημα πριν από το κάλεσμα μιας τάξης ή την προώθηση της εκδήλωσής σας. Για ιδέες σχετικά με το πώς να πραγματοποιήσετε μια απομεμακρυσμένη Ώρα του Κώδικα, διαβάστε τις δικές μας [συμβουλές για μια εκδήλωση εικονικής Ώρας του Κώδικα](https://hourofcode.com/us/how-to/virtual).

### 4. Επιλέξτε μια δραστηριότητα

Παρέχουμε μια ποικιλία διασκεδαστικών εκπαιδευτικών βίντεο καθοδηγούμενα από μαθητές για όλα τα ηλικιακά ζεύγη και όλα τα επίπεδα εμπειρίας. Αν και ενδέχεται να θέλετε να ξεκινήσετε την εκδήλωση με ένα [εμπνευσμένο βίντεο](https://hourofcode.com/us/promote/resources#videos) κοινό για όλους για να δουν όλοι μαζί, να ξέρετε ότι έχει μεγάλη ζήτηση από τους μαθητές η δοκιμαστική χρήση αυτοκαθοδηγούμενων εκπαιδευτικών βίντεο, 

<a href="https://hourofcode.com/us/learn">Εξερευνήστε τις δραστηριότητες</a> και αποφασίστε εκ των προτέρων αν επιθυμείτε ένα μόνο εκπαιδευτικό βίντεο για όλους τους καλεσμένους σας ή αν θα αφήσετε κάθε παΐδι να διαλέξει το δικό του. **Όλες οι δραστηριότητες της Ώρας του Κώδικα** δεν απαιτούν παρά ελάχιστο χρόνο προετοιμασίας από τους διοργανωτές και είναι αυτοκαθοδηγούμενες - επιτρέποντας στους συμμετέχοντες να δουλεύουν σύμφωνα με το δικό τους ρυθμό και με το δικό τους επίπεδο δεξιοτήτων.

[![](/images/tutorials.png)]({{ urls/learn }})

Όταν αποκτήσετε μαι σταθερή πυξίδα για την πορεία της εκδήλωσής σας, μπορείτε μετά να προσκαλέσετε μαθητές, ένα σχολείο ή ακόμα και την ευρύτερη κοινότητα. Σας προτείνουμε να ξεκινήσετε με ένα τοπικό σχολείο με το οποίο συνδέεστε με κάποιο τρόπο ή να [ξεφυλίσσετε το χάρτη μας](https://hourofcode.com/us/map) εκδηλώσεων Ώρας του Κώδικα.

## Συχνές ερωτήσεις Εθελοντισμού

### Δεν έχει επικοινωνήσει μαζί μου κανένας εκπαιδευτικός. Πώς μπορώ να βοηθήσω αλλιώς;

Προσπαθήστε να αναζητήσετε τοπικά σχολεία και τηλεφωνήστε στον διευθυντή/εκπαιδευτικό/γραμματέα και ρωτήστε τους πως μπορείτε να βοηθήσετε.  


### Πόσο διαρκεί μια επίσκεψη στην τάξη;

Μια επίσκεψη στην τάξη με παρουσία συνήθως διαρκεί 60-90 λεπτά ενώ μια επίσκεψη σε τάξη εικονικά συνήθως διαρκεί 20-30 λεπτά.  


### Για ποιες ηλικίες είναι κατάλληλη η Ώρα του Κώδικα;

Ο Code.org και οι συνεργάτες μας σχεδιάζουν όλες τις δραστηρίοτητες της Ώρας του Κώδικα για να καθηλώσουν μαθητές από όλες τις ηλικίες από όλα τα υπόβαθρα. Όλοι - ακόμα και οι ενήλικες - μπορούν να διασκεδάσουν παίζοντας!  


### Πώς επιλέγομαι για τεχνικός εθελοντής ή προσκεκλημένος ομιλητής;

Οι εκπαιδευτικοί θα αναζητούν για εθελοντές στον [χάρτη εθελοντών](/volunteer/local). Προσπαθήστε να έχετε όσο το δυνατόν πιο ολοκληρωμένο προφίλ, ώστε να αυξήσετε τις πιθανότητες να σας διαλέξει ένας εκπαιδευτικός. Αν έχετε κάποια προτίμηση σχετικά με τον τρόπο που θα θέλατε να είστε εθελοντής, όπως να είστε προσκεκλημένος ομιλητής ή τεχνικός υποστήριξης, παρακαλούμε να το υποδείξετε αυτό στην περιγραφή του προφίλ σας.   


Όταν επιλεγείτε, ένας εκπαιδευτικός θα επικοινωνήσει μαζί σας μέσω της πλατφόρμας εθελοντών (δε θα κοινοποιήσουμε ποτέ την διεύθυνση ηλεκτρονικού ταχυδρομείου στον εκπαιδευτικό). Εάν δέχεστε πολλά αιτήματα από εκπαιδευτικούς, μπορείτε να ενημερώσετε τις προτιμήσεις σας, πατώντας τον σύνδεσμο για επεξεργασία των πληροφοριών σας ή για διαγραφή από τη λίστα που βρίσκεται στο τέλος κάθε email από τον εκπαιδευτικό.
