# ඔබේ මුළු පාසල් දිස්ත්‍රික්කයට සම්බන්ධ වන්නේ කෙසේද?

### 2. පාසල් සඳහා නියැදි සැපයුම් සැලැස්මක් සැපයීම

### 4. ඔබගේ දිස්ත්‍රික්කයේ මුල් පිටුවෙහි ප්‍රවර්ධනය කරන්න
