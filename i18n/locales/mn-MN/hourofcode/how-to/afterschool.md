# Сургуулийн дараах хичээл, клубт нэг цагийн кодыг хэрхэн заах вэ

Hour of Code оролдох нь оюутнуудыг компьютерийн шинжлэх ухаантай танилцуулах энгийн бөгөөд хөгжилтэй арга юм, магадгүй хамгийн анх удаа.
