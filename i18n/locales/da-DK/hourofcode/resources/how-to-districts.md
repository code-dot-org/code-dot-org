# Bringe Hour of Code til hele dit skoledistrikt.

## Få alle skoler til at deltage, alle elever skal lære!

  1. **Få fat i de lokale skoler.** Del [ denne e-mail](<%= hoc_uri('/resources#sample-emails') %>) og [ lærerguide](<%= hoc_uri('/resources/how-to') %>), eller inkludere en [ kort omtale](<%= hoc_uri('/resources/stats') %>) i nyhedsbreve.

  2. **Levere en samlet plan for skolerne.** Del denne plan med skolerne for, at give dem ideer om hvordan de kan organisere deres skolers deltagelse. Det er let at gøre det i matematiktimerne, eller skift mellem klasserne, så alle kan bruge computerrummet. I kan indsende jeres egen plan, og har så chance for at vinde $10.000 i hardware på [ http://hourofcode.com/prizes](<%= hoc_uri('/prizes') %>).

  3. **Del på de sociale medier.** Post på [Facebook](https://www.facebook.com/sharer/sharer.php?u=http%3A%2F%2Fhourofcode.com%2Fus) eller [Twitter](https://twitter.com/intent/tweet?url=http%3A%2F%2Fhourofcode.com&text=I%27m%20participating%20in%20this%20year%27s%20%23HourOfCode%2C%20are%20you%3F%20%40codeorg&original_referer=https%3A%2F%2Fwww.google.com%2Furl%3Fq%3Dhttps%253A%252F%252Ftwitter.com%252Fshare%253Fhashtags%253D%2526amp%253Brelated%253Dcodeorg%2526amp%253Btext%253DI%252527m%252Bparticipating%252Bin%252Bthis%252Byear%252527s%252B%252523HourOfCode%25252C%252Bare%252Byou%25253F%252B%252540codeorg%2526amp%253Burl%253Dhttp%25253A%25252F%25252Fhourofcode.com%26sa%3DD%26sntz%3D1%26usg%3DAFQjCNE1GLTUbKZfMlEh9Aj5w0iswz6PYQ&related=codeorg&hashtags=). Eller del en af disse inspirerende [ billeder og citater](<%= hoc_uri('/resources#social') %>) fra verdensledere, eller [ Statistik](<%= hoc_uri('/resources/stats') %>)).

  4. **Annoncer på din kommunes hjemmeside.** Tilføje en af disse [ bannere](<%= hoc_uri('/resources#banners') %>) til din kommunes hjemmeside, for at lade besøgende vide om jeres deltagelse. Link til <http://hourofcode.com>.

  5. **Vær vært for et lokalt Hour of Code arrangement.** Se vores [ guider](<%= hoc_uri('/resources/how-to-events') %>) for hvordan du kan arrangere det, mediemateriale og andet støttemateriale.

  6. **Flere [ præmier](<%= hoc_uri('/prizes') %>) til alle organisatorer.** Hver deltagende underviser vil modtage takkegaver. 100 klasse vil vinde video-chats med særlige gæstetalere. Deltagerne omfatter Bill Gates, Sheryl Sandberg, Ashton Kutcher og mange flere). Og i hver stat i USA, vil én deltagende skole vinde for $10.000 it-udstyr til deres skole!
